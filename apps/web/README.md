# FullBloom Web

React frontend for the FullBloom flower farm CRM.

## Setup

```bash
# Install dependencies
npm install

# Copy env config
cp .env.example .env
# Edit .env with VITE_API_URL (default: http://localhost:8000)

# Start dev server
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS v4
- shadcn/ui components
- Slate + Rose color palette (see constitution)
